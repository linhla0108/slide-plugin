# Ảnh minh hoạ cho hướng dẫn sử dụng

Thư mục này chứa ảnh chụp màn hình dùng trong `docs/how-to-use.md`.

Cách thêm ảnh:

1. Chụp màn hình ứng dụng Claude theo mô tả ở chỗ `<!-- CHÈN ẢNH ... -->`
   trong `docs/how-to-use.md`.
2. Lưu ảnh vào đúng thư mục này với tên gợi ý, ví dụ:
   - `01-man-hinh-chinh.png`
   - `02-tao-slide.png`
   - `03-ket-qua-pptx.png`
   - `04-xem-truoc-component.png`
3. Trong `how-to-use.md`, bỏ phần `<!-- -->` để dòng `![...](...)` hiện ra
   thành ảnh.
