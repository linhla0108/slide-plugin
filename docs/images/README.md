# Illustrative images for the usage guide

This directory contains screenshots used in `docs/how-to-use.md`.

How to add an image:

1. Take a screenshot of the Claude application as described at the
   `<!-- INSERT IMAGE ... -->` markers in `docs/how-to-use.md`.
2. Save the image into this exact directory with a suggested name, for example:
   - `01-man-hinh-chinh.png`
   - `02-tao-slide.png`
   - `03-ket-qua-pptx.png`
   - `04-xem-truoc-component.png`
3. In `how-to-use.md`, remove the `<!-- -->` so that the `![...](...)` line
   shows up as an image.
