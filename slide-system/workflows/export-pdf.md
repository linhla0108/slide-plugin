# Export PDF

1. Prefer browser print from approved HTML for HTML-faithful PDF output.
2. Use a native office renderer when the PDF must represent the PPTX output.
3. Record renderer name, version, font availability, page size, and failures.
4. Verify page count, dimensions, text visibility, image crop, base
   backgrounds, complex overlay images, and z-order.
5. Do not claim parity when the selected renderer cannot run.
