# Legacy Visual Migration

The legacy `component-from-slide/` and `style-from-slide/` inventory is
registered with `qa` status. It is visible for review but unavailable to slide
generation until each item satisfies the publication contract.

Migration decisions:

- `sun-q4-objectives` is categorized as a template.
- Existing names remain available through `aliases.json`.
- `#3531FF` is recorded as a legacy source value. The canonical SUN.STUDIO blue
  remains `#3333FF`.
- Legacy source files stay in place until all references are inventoried.
- Existing reports and phase manifests are not rewritten.
