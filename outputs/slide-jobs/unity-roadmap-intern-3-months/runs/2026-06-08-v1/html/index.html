Tháng 3

Làm chủ tính năng

Mục tiêu không chỉ là code xong, mà là biết bóc tách, phối hợp QA, polish và bàn giao có trách nhiệm.

Tuần 1

Planning

Tự lên kế hoạch và bóc tách logic cho Shop hoặc Daily Quest, sau đó xác nhận với Mentor.

Tuần 2

Execution

Tự code độc lập, ghép UI/Animation bằng DOTween và báo cáo Daily Stand-up.

Tuần 3

Polish

Bàn giao QA test, tối ưu code và đánh bóng game feel trước khi chốt.

Tuần 4

Graduation

Viết tài liệu bàn giao, thuyết trình hành trình và phỏng vấn lên Fresher/Junior.
