# Slide Jobs

Each job uses:

```text
<job-id>/
├── requirements/
├── inputs/
└── runs/<run-id>/
    ├── analysis/
    ├── content/
    ├── plan/
    ├── slide-source/html/
    ├── assets/
    ├── exports/html/
    ├── exports/pptx/editable/
    ├── exports/pptx/ppt-master/
    ├── exports/pptx/image-fallback/
    ├── exports/pdf/
    ├── qa/
    ├── reports/
    └── manifest.json
```
